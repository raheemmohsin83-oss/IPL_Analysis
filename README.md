# 🏏 IPL Data Analysis System

A simple Python-based data analysis project that explores IPL (Indian Premier League) match data using **Pandas, Matplotlib, and Seaborn**.
The system provides an interactive menu to visualize and analyze team performance.

---

## 🚀 Features

* 📊 Total Runs by Team (Bar Chart)
* 📈 Runs per Over (Line Graph)
* 📉 Run Distribution (Histogram)
* 🥧 Top Teams Visualization (Pie Chart)
* 📦 Boxplot of Runs (Statistical Overview)
* 🔁 Menu-driven interactive system

---

## 🧠 Technologies Used

* Python 🐍

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/ipl-data-analysis.git
```

2. Navigate to the folder:

```
cd ipl-data-analysis
```

3. Install dependencies:

```
pip install pandas matplotlib seaborn
```

4. Run the script:

```
python ipl_analysis.py
```

---

## 🎮 Usage

After running the script, a menu will appear:

```
===== IPL ANALYSIS MENU =====
1. Total Runs by Team
2. Runs per Over
3. Run Distribution
4. Top Teams
5. Boxplot
6. Exit
```

Enter a number to view the corresponding analysis.
